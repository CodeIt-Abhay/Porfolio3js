# Abhay Pratap - Interactive 3D Developer Portfolio [![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen)](https://abhaypratap1.vercel.app/)

Welcome to my personal 3D developer portfolio project! This application showcases my academic credentials, professional internships, and featured projects in a fully responsive, animated, and premium 3D workspace.

## 🔗 Live Site
Explore the live, interactive portfolio here: **[https://abhaypratap1.vercel.app/](https://abhaypratap1.vercel.app/)**

---

## 🚀 Key Features
- **3D Astronaut Rig**: Interactive Three.js / React Three Fiber rendering of a floating astronaut that responds to mouse cursor movement.
- **Dynamic Capsule Navigation**: Premium desktop navigation bar featuring a sliding pill/capsule backdrop indicator (powered by Framer Motion).
- **Glassmorphic Grid System**: Modern, responsive layout displaying my details, time-zone, email copier, drag-and-drop skills card stack, and orbiting framework logos.
- **Real Timeline Experience**: Interactive timeline showcasing my recent internships.
- **Education & Credentials Panel**: Fully custom section showcasing my B.Tech details, certifications (NPTEL, CCNA), and achievements.
- **Interactive Contact Form**: Integrated with EmailJS for direct, client-side email delivery.

## 💻 Local Setup and Installation

Follow these steps to run the project locally on your machine:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/CodeIt-Abhay/Porfolio3js.git
   cd Porfolio3js
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the local development server**:
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` (or the port specified in terminal) in your browser.

4. **Build for production**:
   ```bash
   npm run build
   ```
   The compiled assets will be bundled into the `/dist` directory.
